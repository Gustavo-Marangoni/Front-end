Linguagem de Programação C

A linguagem de programação C foi criada por Dennis Ritchie
no ano de 1972, nos laboratórios da Bell Labs.
Ela é uma linguagem de médio nível, muito poderosa e eficiente.

História

A linguagem C foi desenvolvida para melhorar a linguagem B.
Ela foi criada para ajudar no desenvolvimento do sistema operacional Unix.
Até hoje, C é muito utilizada para sistemas operacionais e softwares de baixo nível.

Principais Características

- Linguagem compilada
- Alta performance
- Controle direto de memória
- Portabilidade entre sistemas
- Base para outras linguagens como C++, C# e Java

Exemplo de Código em C

#include <stdio.h>

int main() {
    printf("Olá, Mundo!");
    return 0;
}

Vantagens

C é rápida e eficiente.
É muito usada em sistemas embarcados.
Permite acesso direto à memória.

Desvantagens

Não possui coleta automática de lixo.
Pode ser difícil para iniciantes.
Erros de memória podem causar falhas graves.

Conclusão

A linguagem C é uma das linguagens mais importantes da história da programação.
Muitas linguagens modernas foram baseadas nela.
Aprender C ajuda a entender melhor como os computadores funcionam.
